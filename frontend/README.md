# Frontend - Portafolio de Keiver Quevedo

Esta carpeta contiene el frontend moderno del portafolio (SPA) construido con **React 19 + Vite 8** (stack actual 2026).

## Cómo iniciar

```bash
cd frontend
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notas

- Reemplaza `public/avatar.svg` por tu foto real (misma ruta o actualiza el código para usar otro nombre).
- Los logos de las empresas son SVG de placeholder en `public/companies/`.
- La sección de contacto actualmente usa un `alert()` en el formulario; integra tu backend (Supabase, email, etc.) donde corresponda.
