# Proyecto Contable

Estructura del proyecto contable con panel de administración, configuración y despliegue.

## Carpetas principales

- `.github/workflows` - CI/CD
- `.vscode` - ajustes de editor
- `css` - hojas de estilo
- `js` - lógica de frontend
- `admin` - páginas del panel administrativo
- `sql` - scripts SQL

## Archivos de entorno
- `.env` (no subir)
- `.env.example`

## Uso
1. Copia `.env.example` a `.env` y rellena variables.
2. `npm install` para dependencias.
3. `npm start` para servidor local.

> Recuerda mantener `js/config.js` fuera del control de versiones.
